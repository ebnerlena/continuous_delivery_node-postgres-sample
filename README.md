# Node PostgreSQL Sample

A simple Express app using Sequelize to access PostgreSQL.

## Usage

- Run `npm install`
- Start a PostgreSQL database
- Set the environment variables for DB_HOST, DB_NAME, DB_USER, DB_PASSWORD
- Run `node ./server.js`
